　　　　　　One Chip Board「メスイサ」復刻版

One Chip Board「メスイサ」復刻版は当時一世を風靡したあのメスイサ(MeSX)を【唯一無二の俺仕様🔥】で復刻した新感覚の8ビットOne Chip Boardです。

One Chip Board「メスイサ」復刻版仕様

CPU Z80 3.58Mhz相当(高速モード付き)
ROM BIOS/BASIC 32KB
RAM マッパーメモリー384KB
ビデオ TMS9918相当(VGA 31KHz出力)
音声 AY-3-8910相当/PPI 1ビットサウンド相当
キーボード 日本語USBキーボード対応
カートリッジスロット 基本スロットx2
ATARIポート x2

MSXπシリーズとの互換性について

MSX交響楽団を含め、各機能概ね大丈夫だと思います。

IOEμも使えるよ。

MSX(MSX1)との互換性について

以下の制限があります。ゲームカセット類はまず大丈夫だと思いますがMSXπ以外の機能拡張カートリッジについては動かないものが色々あると思います。

* 一部のスロット信号を省略しています
リフレッシュ信号を使用したDRAM拡張カートリッジや割り込み信号を使用した機能拡張カートリッジ等(RS232CやMSX➋代等)は動作しません。
※MSXπシリーズは影響を受けません。

* 正負12V電源を省略しています
FMPACやRS232Cカートリッジは動作しません。
※MSXπシリーズは影響を受けません。

* Z80命令の処理時間が実機のZ80のそれと大きく異なります
Z80エミュレーターを間欠動作(秒間31500回)させる事により表面上の動作速度を3.58Mhz相当にしています。
命令処理時間を利用してアクセス間隔を空ける必要のある機能拡張カートリッジは不具合を起こす場合があります。
※YM2413はこれに該当しますがYM2413に限り例外的に対応を加えています。MSXπシリーズは影響を受けません。

* ATARIポートは標準的な2ボタンジョイススティックのみサポート
マウスや特殊デバイスは動作しません。

高速モードについて
高速モードをONにするとZ80処理速度の調整機能が無効になりZ80エミュレーターが全力動作します。
VDP機能やPSG機能は標準速度で動作するので高速モードでも多くのゲームが普通に動作します。
ヒント👉
高速モードONでもスロットアクセスは高速になりません。よってRAMやプロクラムがスロット上にある場合は高速モードでもあまり速くなりません。
メスイサ内蔵RAM上にプログラムがある場合はかなり高速に動作します。MSXπでNextorを使用する場合はRAM無しのsunriseを使用すると良いです。

電源について
外形5.5mm内径2.1mmセンタープラスのプラグを持つDC5VのACアダプターを使用して下さい。出力電流は1Aもあれば十分です。
安価なUSB変換ケーブルもあるようです👉
カモン DC-5521 USB-DC電源ケーブル A(オス)-DCプラグ(外径5.5mm内径2.1mm)ストレート端子 1.2m

キーボードについて

RP2350Bボード上のUSBタイプCコネクターに日本語USBキーボードを接続して下さい。タイプC->タイプA変換アダプターはダイソーにもあります。

ヒント👉
ゲームでよく使われる「斜め方向+スペース」のような組み合わせの同時押しに対応していないキーボードがあります。

キーボードのLEDについて

    NUMLOCK 消灯:高速モードOFF(デフォルト) 点灯:高速モードON
    CAPSLOCK CAPS LED
    SCROLLLOCK かなLED

音声出力について
音声アンプ類は一切搭載していません。アンプ付きスピーカーを使用して下さい。
音声はモノラルですが出力ジャックはステレオ仕様です。ステレオプラグを挿して使用して下さい。モノラルプラグは挿さないで下さい(音出ません)。
VGAコネクタからは音声出力されません念の為💦。
